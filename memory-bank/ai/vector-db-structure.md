# Vector DB Structure

VectorDB хранит эмбеддинги для:

- документации Memory Bank;
- ключевых файлов кода;
- схем БД;
- описаний API;
- REDMAP;
- лора мира.

Основные папки эмбеддингов:
- devos
- cybernation
- backend
- redmap
- api
- db
- social
- hacking
- corp
- labs
- blueprints

Каждый документ или важный файл кода имеет:
- id (путь)
- embedding (вектор)
- метаданные (тип, слой REDMAP и т.п.)
