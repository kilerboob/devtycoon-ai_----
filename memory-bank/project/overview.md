# DeVOS / CyberNation — Project Overview

DeVOS — внутриигровая кибер-операционная система, работающая как полноценный рабочий стол,
браузер, IDE и инфраструктура для MMO-мира CyberNation.

CyberNation — цифровая планета с корпорациями, лабораториями, гильдиями, студиями, игроками,
рынками и кибервойнами. Проект сочетает:

- OS-интерфейс (окна, таскбар, приложения DeVOS)
- Экономику (рынки, контракты, чертежи, крафт)
- Социальные системы (чаты, звонки, угрозы, гос-сообщения)
- Хакерские механики (взлом лабораторий, игроков, NPC)
- Мировую карту (Planet Sphere с зонами влияния корпораций и гильдий)

Все изменения и разработка должны следовать REDMAP v6.0 и правилам AI Orchestrator / AI Build Crew.
